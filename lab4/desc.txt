seidel_solver(A, b, epsilon, itmax)
Эта функция реализует метод Зейделя (Gauss-Seidel method) для решения системы линейных уравнений Ax = b.
A - матрица коэффициентов системы уравнений.
b - вектор правых частей уравнений.
epsilon - заданная точность для окончания итераций.
itmax - максимальное число итераций.
Функция возвращает приближенное решение x, количество итераций it, и норму разности двух последних итераций для x.

relaxation_solver(A, b, omega, epsilon, itmax)
Эта функция реализует метод релаксации (Successive Over-Relaxation или SOR method) для решения системы линейных уравнений Ax = b.
A, b, epsilon, и itmax имеют те же значения, что и в seidel_solver.
omega - параметр релаксации, который используется для ускорения сходимости метода Зейделя.
Функция возвращает приближенное решение x, количество итераций it, и норму разности двух последних итераций для x.

optimal_omega(A, b, epsilon, itmax, omega_start=1.0, omega_end=1.2, omega_step=0.01)
Эта функция находит оптимальное значение параметра релаксации omega для метода релаксации, применяемого к системе уравнений Ax = b.
Она ищет оптимальное omega в диапазоне от omega_start до omega_end с шагом omega_step.
Для каждого значения omega в диапазоне, функция вызывает relaxation_solver и сравнивает количество итераций, необходимых для достижения заданной точности epsilon.
Функция возвращает значение omega, которое обеспечивает наименьшее количество итераций.

"""

Вот некоторые особенности выбора интервала для ω:
ω = 1: Это значение соответствует методу Зейделя, где текущее приближение не зависит от предыдущего.
ω > 1: Параметр ω, больший чем 1, может ускорить сходимость, потому что он позволяет методу релаксации "пропустить" предыдущее приближение и сильнее учитывать текущий шаг.
ω < 1: Если ω меньше 1, метод может сходиться медленнее, но может быть более устойчив к невязкам и ошибкам вычисления.
